# Nvim Flash

Augustine's Neovim dotfiles

## Features

1. Git Integration
2. Syntax Highlighting
3. Nvim-Tree
4. Telescope
5. Autopairs
6. Autocompletion

## Supported Languages

1. Javascript
2. Typescript
3. Python
4. Golang
5. Dart
6. Yaml

## Installation 

```sh
>>> git clone https://github.com/The-Flash/nvim-flash.git
```

Move the contents of nvim-flash to "~/.config/nvim"
